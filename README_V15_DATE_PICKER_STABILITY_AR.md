# V15 - B Item Date Picker Stability Fix

التعديل ده مخصص لحل مشكلة اختفاء الكالندر في B Item Control.

## الملفات المطلوب رفعها
- bitem.html فقط

## لا تلمس
- functions
- D1
- SQL
- باقي صفحات الموقع

## التعديل
- إيقاف فتح التعديل من خلية التاريخ نفسها.
- التعديل يفتح فقط من زر Edit الخاص بنفس الصف.
- عند الضغط على Edit يظهر حقل التاريخ ثابت ولا يختفي مع أي Refresh داخلي.
- تم إضافة زر 📅 بجانب حقل التاريخ لفتح الكالندر يدويًا.
- زر X يلغي التعديل بدون إعادة تحميل الجدول.
- زر ✓ يحفظ ثم يحدث حالة B Item.
