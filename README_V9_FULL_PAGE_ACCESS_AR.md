# V9 Full Page Access Matrix

هذه النسخة تجعل الصلاحيات مبنية على صفحتين لكل مستخدم:

- View Access: الصفحات التي تظهر في القائمة ويمكن فتحها.
- Edit Access: الصفحات التي يسمح للمستخدم بالتعديل داخلها.

إذا لم يتم اختيار View لصفحة معينة، فلن تظهر في القائمة، ولو المستخدم كتب رابطها مباشرة سيتم منعه.

الصفحات الحالية داخل نظام الصلاحيات:

- Test Pack Dashboard
- B Punch Edit / Control
- B Item Monitoring
- User Management

ملاحظات:

- Admin يظل Full Access كحماية للنظام.
- User / Editor يمكن تحديد View و Edit لكل صفحة متاحة للتعديل.
- Viewer يمكن تحديد الصفحات التي يشاهدها فقط، بدون تعديل.
- يمكن عمل Clear All لأي User/Viewer بحيث لا تظهر له أي صفحة تشغيلية.
- لا تعمل SQL Reset ولا تحذف D1.
